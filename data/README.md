# Data

This folder contains the experimental datasets associated with the earthen embankment sensing studies.

## dataset-1
* This dataset contains measurements from a lab-scale earthen embankment experiment with nine wireless sensing spike packages arranged in a grid to monitor water flow, seepage, and saturation changes through the embankment.
* Dataset initially associated with the MDPI paper: Sydney Morris, Puja Chowdhury, Malichi Flemming, Ayman Mokhtar, Austin R. J. Downey, Jasim Imran, and Sadik Khan. Detecting piping failure in levees using soil saturation mapping and a wireless sensor network. Submitted to Infrastructures for possible open access publication, 2026.

## dataset-2
* This dataset contains measurements from a lab-scale earthen embankment experiment with multiple sensing spikes arranged in a row to monitor wetting and drying behavior, including how moisture remained in the embankment during the drying cycle.
* Dataset initially associated with the ASME paper: Sydney Morris, Ayman Mokhtar, Malichi Flemming, Austin R. J. Downey, Puja Chowdhury, Matthew Burnett, Jasim Imran, and Sadik Khan. Self-contained electrical conductivity sensing spikes for monitoring of levee wetting and drying cycles. In Volume 5: 21st IEEE/ASME International Conference on Mechatronic and Embedded Systems and Applications (MESA), IDETC-CIE2025. American Society of Mechanical Engineers, August 2025. doi:10.1115/detc2025-169136
